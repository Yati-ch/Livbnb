# 🏡 Livbnb – Full Stack Web Application

Livbnb is a **full‑stack web application** inspired by Airbnb, built to demonstrate real‑world web development skills such as authentication, RESTful APIs, database modeling, and MVC architecture. This project highlights my ability to design, develop, and deploy scalable web applications using modern technologies.

---

## 🎯 Project Objective

The objective of Livbnb is to create a platform where users can explore, create, and manage property listings while ensuring secure authentication and a smooth user experience. This project was developed as part of my learning journey in full‑stack development and reflects industry‑level coding practices.

---

## 🚀 Key Features

* 🔐 **User Authentication & Authorization** (Login / Signup / Logout)
* 🏠 **Property Listings (CRUD)** – Create, Read, Update, Delete
* 🖼️ **Image Upload & Storage**
* 📍 **Location‑based Property Information**
* ⭐ **Reviews & Ratings System**
* 🧭 **RESTful Routing & MVC Architecture**
* 📱 **Responsive UI** for mobile & desktop

## ⚙️ Installation & Setup

### Prerequisites

* Node.js
* MongoDB
* Git

### Steps to Run Locally

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/livbnb.git
   ```

2. **Navigate to the project folder**

   ```bash
   cd livbnb
   ```

3. **Install dependencies**

   ```bash
   npm install
   ```

4. **Configure Environment Variables**

Create a `.env` file in the root directory:

```
MONGODB_URL=your_mongodb_connection_string
SESSION_SECRET=your_secret_key
CLOUDINARY_API_KEY=your_key
CLOUDINARY_SECRET=your_secret
```

5. **Start the server**

   ```bash
   npm start
   ```

6. Open browser and visit:

```
http://localhost:8080
```
